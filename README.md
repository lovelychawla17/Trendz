# Trendz – Full-Stack E-commerce Website

Full-stack e-commerce web application demonstrating frontend and backend development skills.

## Live Demo
- Frontend: https://trendz-store.netlify.app
- Backend API: https://trendz-backend-8smj.onrender.com  

## Tech Stack
- HTML, CSS, JavaScript
- Node.js, Express.js
- MySQL
- Netlify, Render and Railway

## Features
- User Signup & Signin
- REST API development
- MySQL database integration
- Add to Cart functionality

## Project Structure
frontend/   – UI files (HTML, CSS, JS)
backend/    – Node.js + Express API
database/   - MySQL

## Run Locally
cd backend
npm install
npm start
